# Task manager - todo 
The application is todo list. Here it has possibility to create, modify, delete your business 

### Technologies
- Spring boot 3.1
- Spring JPA
- Spring thymeleaf
- Bootstrap 5.2.3

### How to start
- use docker-compose

### How to build docker image

- docker build --tag=[name]:[tag] .
- docker push [name]:[tag]
